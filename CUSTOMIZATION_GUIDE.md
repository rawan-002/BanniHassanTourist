# دليل التخصيص — موقع بني حسن السياحي

هذا الدليل يشرح كيفية إجراء التعديلات البصرية الأساسية على الموقع لضمان توافقه مع الهوية المؤسسية.

---

##  نظرة عامة على ملفات الإعداد
لإجراء أي تعديل، ستحتاج للتعامل مع الملفات التالية حسب نوع التعديل المطلوب:

| العنصر المراد تغييره | الملف المسؤول | المسار |
|:---|:---|:---|
| **الألوان والخطوط البرمجية** | `theme.config.js` | `src/presentation/theme/` |
| **ألوان الـ Navbar والفوتر (CSS)** | `components.css` | `src/presentation/components/` |
| **الشعارات والصور** | مجلد الـ `assets` | `src/assets/` |

---

##  1. تعديل الألوان الرئيسية (ملف JS)
يتم التحكم في الألوان الأساسية للأزرار والتفاعلات عبر ملف التكوين.
**المسار:** `src/presentation/theme/theme.config.js`

```javascript
colors: {
  primary:   '#4D3873',   // اللون البنفسجي الرئيسي (الأزرار والعناوين)
  secondary: '#2F8872',   // اللون الأخضر الثانوي
  accent:    '#DA943B',   // اللون الذهبي للتمييز

  // تدرج شريط التنقل (Navbar)
  navbarGradient: 'linear-gradient(90deg, #3e2e75 0%, #e6a14c 50%, #1d5075 100%)',

  // أزرار قسم الترحيب (Hero Section)
  heroButtonBg:    'rgba(159, 165, 212, 0.3)', // أزرار شفافة
  heroButtonHover: '#868fdb',                  // عند التحويم بالماوس

  // ألوان تذييل الصفحة (Footer)
  footerBg: 'linear-gradient(180deg, rgb(28, 27, 57) 0%, rgb(37, 37, 89) 50%, rgb(43, 43, 98) 100%)',
}
